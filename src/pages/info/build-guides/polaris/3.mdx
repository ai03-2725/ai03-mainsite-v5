    ## Preparing the case
    Let's begin with the case.
    
    1. Unbox the keyboard and all related parts if not already done.

    2. Of the two supplied hex keys, use the larger one to unscrew the four weight screws.
      ![Weight screws](/images/build-guides/polaris/polaris-build-guide-3-2.png)

    3. Carefully remove the weight, and set aside.
      ![Weight removal](/images/build-guides/polaris/polaris-build-guide-3-3.png)

    4. Use the smaller of the two hex keys to unscrew the eight case screws.
      ![Case screws](/images/build-guides/polaris/polaris-build-guide-3-4.png)

    5. Carefully remove the bottom case half, and set aside.
      ![Bottom removal](/images/build-guides/polaris/polaris-build-guide-3-5.png)

    6. At this point, the case should be fully disassembled.
        Prepare the plate and adhesive gaskets.
        ![Plate](/images/build-guides/polaris/polaris-build-guide-3-6.png)

    7. Carefully place 6 of the long gaskets onto the plate, one per tab.  
      **Note:** For the minority who are planning on using aftermarket custom plates, install these gaskets on the case top area where the plate tabs overlap. For specific warnings and instructions, see [here](#).
      ![Gasket locations](/images/build-guides/polaris/polaris-build-guide-3-7-1.png)
      ![Gasket install](/images/build-guides/polaris/polaris-build-guide-3-7-2.png)

    8. Repeat on the underside; there should be 12 gaskets installed at the end of this step.
      **Note:** As per the previous step, if you are planning on using aftermarket custom plates, place the gaskets on the bottom case half where the plate tabs overlap.
      ![Plate gaskets underside](/images/build-guides/polaris/polaris-build-guide-3-8.png)

    9. Set the completed plate aside.
      Prepare the case top half and the small rectangular gaskets.
      ![Top](/images/build-guides/polaris/polaris-build-guide-3-9.png)

    10. Place the small gaskets horizontally at roughly the points indicated, on the inner wall of the case.
      ![Vertical gasket locations](/images/build-guides/polaris/polaris-build-guide-3-10-1.png)
      They should be placed at “plate-height” on the center band, flush to the lower edge as shown.
      ![Vertical gasket install](/images/build-guides/polaris/polaris-build-guide-3-10-2.png)

    11. Set the completed top case half aside.Prepare the case bottom half.
      Install the provided rubber feet into the insets. 
      ![Rubber feet](/images/build-guides/polaris/polaris-build-guide-3-11.png)  
      The case is now ready for assembly.

    <br />
    
    Now it is time to solder the switches.
